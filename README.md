 # 主な機能
1. **ダッシュボード（ホーム）**
 * **総資産と推移:** 現在の総残高と本日の増減をひと目で確認できます。また、過去の資産推移を折れ線グラフで視覚化しています。
 * **口座内訳と目標:** メイン口座、貯蓄、お財布などの内訳を円グラフで表示。設定した目標（ハワイ旅行など）に対する進捗ゲージも確認できます。
2. **直感的な記録モーダル**
 * **3つの取引タイプ:** 支出・収入・口座間の「振替」に対応しています。
 * **クイック入力:** 金額入力時に「+1千」「+5千」「+1万」のショートカットボタンを使って素早く入力できます。横スクロール対応のUIで、カテゴリーや口座もワンタップで選択可能です。
3. **口座・財布の管理（残高調整）**
「口座・財布」タブから、各口座の残高を直接編集できます（鉛筆アイコンをクリック）。実際の残高とのズレを手動で簡単に修正可能です。
履歴管理
「履歴」タブから全取引のリストを確認できます。不要な記録や間違えた記録（ゴミ箱アイコン）を削除すると、紐づく口座の残高も自動的に計算し直されて元に戻ります。


 #  技術スタック（Tech Stack）
モダンなフロントエンド技術を組み合わせて構築しており、高速で滑らかな動作を実現しています。
  * **コアフレームワーク:** React 18 (TypeScript、Vite)
  * **スタイリング:** Tailwind CSS （レスポンシブデザインとモダンなUIコンポーネントの構築）
  * **アニメーション:** 動作/反応 (Framer Motion) （モーダルの開閉やタブの切り替えを滑らかに表現）
  * **データ視覚化:** チャートの再作成 （資産推移の折れ線グラフと、口座内訳の円グラフ）
  *  **アイコン:** ルシド・リアクト （シンプルで一貫性のあるベクターアイコン）
# 使い方（How to Use）
1.  **お金を記録する:**
パソコンなら右上の**「記録する」ボタン、スマートフォンなら画面下の「＋」**ボタンを押します。
モーダルが開いたら、「支出」「収入」「振替」のタブを選びます。
「Amount（金額）」を入力し（+1千などのボタンも便利です）、対象の口座やカテゴリーを選んで「記録」ボタンを押すだけです。
2.  **残高を微調整する:**
下部（または上部）メニューから**「口座・財布」**タブを開きます。
修正したい口座の残高の横にマウスを合わせる（スマホの場合はタップ・長押し付近）と表示される鉛筆アイコンを押します。
現在の実際の残高を入力し、チェックマークを押すと即座に反映されます。
3. **間違えた記録を取り消す:**
**「履歴」**タブを開きます。
削除したい記録の右端に表示されるゴミ箱アイコンを押します。
削除が完了すると、その支出・収入・振替が行われる前の状態に口座残高が自動で戻ります。
